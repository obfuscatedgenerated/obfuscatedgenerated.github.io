      Registers a key event handler. Handlers with no filter run BEFORE filtered handlers.

      Parameters

      • handler: KeyEventHandler

        The handler to register

      • props: {
            keyString?: string;
            domEventCode?: string;
            block?: boolean;
            high_priority?: boolean;
        }

        The properties of the handler. Key is the key as a string to filter by, domEventCode is the DOM event code to filter by. Block determines whether the event should be blocked from bubbling up to following handlers and/or the terminal display. High priority determines whether the handler should be placed at the beginning of the handler list.

      Returns () => void

      A function to unregister the handler
